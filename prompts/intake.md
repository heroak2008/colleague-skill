# 基础信息录入脚本

## 开场白

```
我来帮你创建影分身 Skill。只需要回答 3 个问题，每个都可以跳过。
```

---

## 问题序列

### Q1：影分身代号

```
你的影分身叫什么？（花名、昵称或代号都行，多个字用 - 连接）

例：qing-yun
```

- 接受任意字符串
- 生成的 slug 统一用 `-` 连接（不用下划线）
- 中文自动转拼音再用 `-` 连接（"青云" → `qing-yun`，"小李" → `xiao-li`）
- 英文直接小写 `-` 连接（"Tech Writer" → `tech-writer`）

---

### Q2：背景信息

把职业、领域、写作方向放在一个问题里，让用户一句话说完：

```
用一句话描述你的背景——职业、擅长领域、写作或内容方向，想到什么写什么，跳过也行。

例：后端工程师 专注分布式系统 写技术博客
```

从用户的回答中解析以下字段（缺失的留空）：
- **职业**
- **领域**（可以是多个，逗号分隔）
- **写作/内容方向**（若有）

---

### Q3：风格画像

把 MBTI、个性标签、领域文化标签、自我印象全部合在一起，让用户自由描述：

```
用一句话描述你的风格——MBTI、个性特点、文化烙印、你对自己的印象，
想到什么写什么，跳过也行。

例：INTJ 直接犀利 字节味 写文章喜欢结论先行，爱怼烂方案
```

从用户的回答中识别并提取以下字段（缺失的留空）：
- **MBTI**：16 种标准类型
- **个性标签**：从下方标签库匹配，也接受自定义描述
- **文化标签**：从下方标签库匹配
- **自我印象**：无法归类的自由描述，直接保留原文

#### 个性标签库

**工作态度**：认真负责 / 差不多就行 / 完美主义 / 拖延症 / 结果导向

**沟通风格**：直接犀利 / 绕弯子 / 话少 / 话多 / 爱发语音 / 只读不回 / 秒回强迫症

**决策风格**：果断 / 反复横跳 / 独立判断 / 数据驱动 / 全凭感觉 / 爱问本质

**表达习惯**：结论先行 / 娓娓道来 / 爱用列表 / 爱举例子 / 爱讲大道理 / 引经据典

#### 文化标签库

- **字节范** — 坦诚直接、追求 impact、开口必讲 context、爱说"对齐"
- **阿里味** — 六脉神剑、爱用"赋能""抓手""生态""闭环"
- **腾讯味** — 数据说话、赛马机制、克制保守、注重用户体验
- **华为味** — 奋斗者文化、流程规范、爱做 PPT 汇报、强调执行力
- **百度味** — 技术至上、层级意识强、内部竞争激烈
- **美团味** — 极致执行、抠细节、本地化思维
- **第一性原理** — 追问本质、拒绝类比、激进简化
- **OKR 狂热者** — 凡事先定义 Objective、KR 要量化
- **创业公司派** — 全栈思维、资源有限、结果导向、容忍混乱

## 确认汇总

收集完毕后展示：

```
信息汇总：

  🥷  {代号}
  💼  {职业}（若未填则省略）
  🎯  领域：{领域列表}（若未填则省略）
  ✍️   写作方向：{方向}（若未填则省略）
  🧠  {MBTI}（若未填则省略）
  🏷️   个性：{标签列表}（若未填则省略）
  🏢  文化：{标签列表}（若未填则省略）
  💬  印象：{印象文本}（若未填则省略）

确认无误？（确认 / 修改 [字段名]）
```

用户确认后进入 Step 2 文件导入。
